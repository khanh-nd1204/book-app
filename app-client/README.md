## 1. Prerequisites

Đảm bảo bạn đã cài đặt các công cụ sau:

- ✅ [Node.js (v18+)](https://nodejs.org/)
- ✅ [Yarn](https://classic.yarnpkg.com/lang/en/) hoặc [npm](https://www.npmjs.com/)
- ✅ [VS Code](https://code.visualstudio.com/)

## 2. Tech Stack
Dự án sử dụng các công nghệ sau:

- **Vite**
- **React 18+**
- **Ant design** 
- **Redux Toolkit** 
- **React Router DOM**
- **Axios**
- **ESLint + Prettier**

## 3. Run & Development

### Cài đặt dependencies

```
# Dùng yarn
$ yarn install

# Hoặc dùng npm
$ npm install
```

### Khởi chạy ứng dụng

```
# Dùng yarn
$ yarn dev

# Hoặc npm
$ npm run dev
```

## 4. Build Production

```
# Dùng yarn
$ yarn build

# Hoặc npm
$ npm run build
```
